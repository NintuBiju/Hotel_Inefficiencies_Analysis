🏨 Hotel Operations Efficiency Analysis
A data analytics project identifying inefficiencies in hotel room service and operations across 2019–2024.

Overview
This was my first end-to-end data analytics project, built entirely in Excel. The goal was to dig into hotel operational data and surface the root causes of inefficiency — from slow task completion to complaint spikes — using a structured, multi-angle analysis approach.

Dataset

Period covered: 2019 – 2024
Tool used: Microsoft Excel (data cleaning, analysis, visualization)
Scope: Room service requests, guest complaints, occupancy rates, employee performance, and external factors across multiple hotel locations


Project Structure
The project is organized into 12 analysis sections:

Summary Statistics — Overall operational metrics: service requests, completion times, complaints, occupancy
Service Request Volume Analysis — Request trends by month, year, and service type
Seasonal Trend Analysis — Peak vs. low seasons and their impact on workload
Task Completion Analysis — Average completion times, delayed tasks, and bottlenecks
Guest Experience Analysis — Satisfaction scores and complaint patterns
Employee Performance Analysis — Productivity metrics and task completion efficiency by employee
External Factors Analysis — Impact of weather and special events on operational demand
Resource Utilization Analysis — Staffing efficiency and workload distribution
Correlation Analysis — Relationships between occupancy, requests, complaints, and satisfaction
Trend and Anomaly Detection — Unusual request spikes and operational irregularities
Geographic Analysis — Performance comparison across hotel locations
Executive Summary Dashboard — Key KPIs and actionable recommendations


Key Findings

Higher occupancy directly drove higher service request volumes
Longer task completion times correlated with lower guest satisfaction scores
Certain seasons generated significantly more workload than others
Employee performance varied noticeably — some staff consistently outperformed peers
Weather and special events created measurable surges in operational demand
Some hotel locations experienced disproportionately more complaints and delays
Resource allocation was often misaligned with actual demand patterns


Process

Data Cleaning — Handled missing values, standardized formats, removed duplicates
Exploratory Analysis — Understood distributions, outliers, and baseline trends
Diagnostic Analysis — Investigated root causes behind inefficiency patterns
Visualization — Created charts and dashboards to communicate findings clearly
Insight Extraction — Pulled actionable takeaways from all 12 analysis sections
